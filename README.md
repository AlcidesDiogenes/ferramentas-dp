# Ferramentas DP

O **Ferramentas DP** é uma plataforma desenvolvida para centralizar, automatizar e otimizar rotinas administrativas, fiscais e de Departamento Pessoal. O projeto foca em reatividade, conformidade com a legislação trabalhista e eficiência no manuseio de dados (planilhas e PDFs).

## 🚀 Sobre o Desenvolvimento
Este projeto é um exemplo de **desenvolvimento assistido por IA**. Todo o código, arquitetura de componentes, lógica de cálculo baseada na CLT e estilização foram desenvolvidos em colaboração direta com Inteligência Artificial, garantindo uma estrutura modular, escalável e de fácil manutenção.

## 🛠️ Tecnologias Utilizadas
- **Frontend:** HTML5, CSS3 (Modular e Responsivo), JavaScript (ES6+).
- **Processamento de Dados:** [SheetJS](https://sheetjs.com/) (para manipulação de planilhas).
- **Geração de PDF:** [pdfMake](http://pdfmake.org/) (para espelhos de ponto e relatórios).
- **Arquitetura:** Design System próprio focado em usabilidade profissional.

## 📂 Funcionalidades Principais
- **Gestão de Jornada:** Importação de planilhas, cálculo automático de horas extras, adicionais noturnos e apuração de folgas.
- **Simuladores de Cálculos:** Ferramentas para cálculos de Pró-labore, INSS e IRRF.
- **Auditoria:** Mapeamento de afastamentos e análise de passivos fiscais via PDF.
- **Importação TXT:** Conversor otimizado para o leiaute da Domínio Sistemas.

## 📱 Responsividade
O sistema é totalmente adaptável, oferecendo uma interface limpa em desktops (blocos lado a lado) e uma navegação otimizada em dispositivos móveis (listas verticais para melhor legibilidade).
